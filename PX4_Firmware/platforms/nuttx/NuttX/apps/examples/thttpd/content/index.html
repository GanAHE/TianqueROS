These web pages are served by a port of THTTPD running on top of NuttX. NuttX includes an embedded TCP/IP state that derives from the uIP.

Click on the links above to exercise THTTPD's CGI capability under NuttX. Depending on the configuration, clicking the links will execute the CGI program from either:

  1. An NXFLAT program residing in a NuttX ROMFS file system, or
  2. A built-in application managed by a NuttX BINFS file system.
